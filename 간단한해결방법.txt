간단한 해결 방법
==========================================

Node.js 설치가 어렵거나 시간이 오래 걸리는 경우:

==========================================
옵션 1: Vercel에 직접 배포 (가장 간단)
==========================================

1단계: GitHub에 코드 업로드
   - GitHub 계정이 필요합니다
   - 새 저장소 생성
   - 코드 업로드

2단계: Vercel에 배포
   - https://vercel.com/ 접속
   - GitHub 계정으로 로그인
   - "New Project" 클릭
   - GitHub 저장소 선택
   - 환경 변수 설정:
     * SUPABASE_URL
     * SUPABASE_SERVICE_KEY
     * OPENWEATHER_API_KEY
   - "Deploy" 클릭

3단계: 배포 완료 후 사용
   - Vercel이 제공하는 URL로 접속
   - 모든 기능이 작동합니다!

장점:
   ✅ Node.js 설치 불필요
   ✅ 무료
   ✅ 자동 HTTPS
   ✅ 실제 서버 환경과 동일

==========================================
옵션 2: Node.js 설치 (로컬 개발용)
==========================================

위의 "Node.js설치가이드.txt" 파일을 참고하세요.

==========================================
추천
==========================================

로컬 개발이 꼭 필요하지 않다면:
→ Vercel에 직접 배포하는 것을 강력히 권장합니다!

로컬에서 개발하고 싶다면:
→ Node.js를 설치하고 "Node.js설치가이드.txt"를 따라하세요.
