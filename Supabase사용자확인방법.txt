==========================================
Supabase에서 회원가입한 사용자 확인 방법
==========================================

회원가입 후 Supabase에서 사용자가 제대로 생성되었는지 확인하는 방법입니다.

==========================================
1단계: Supabase 대시보드 접속
==========================================

1. https://supabase.com/ 접속
2. 로그인
3. 프로젝트 선택

==========================================
2단계: 사용자 목록 확인
==========================================

1. 좌측 메뉴에서 "Authentication" 클릭
2. "Users" 탭 클릭 (기본적으로 열려있음)
3. 사용자 목록 확인

==========================================
3단계: 사용자 정보 확인
==========================================

각 사용자 행에서 다음 정보를 확인할 수 있습니다:

- Email: 가입한 이메일 주소
- User UID: 고유 사용자 ID
- Created: 가입 날짜/시간
- Last Sign In: 마지막 로그인 시간
- Confirmed: 이메일 확인 여부
  - ✅ 체크 = 확인됨 (정상)
  - ❌ 빈칸 = 미확인 (이메일 확인 필요)

==========================================
4단계: 사용자 상세 정보 보기
==========================================

1. 사용자 행 클릭
2. 또는 "..." 메뉴 → "View user details" 클릭
3. 다음 정보 확인:
   - User ID
   - Email
   - Email Confirmed (확인 여부)
   - Created At
   - Last Sign In
   - User Metadata (이름 등 추가 정보)

==========================================
5단계: 실시간 확인 방법
==========================================

회원가입을 시도하는 동안:

1. Supabase "Authentication" → "Users" 페이지 열기
2. 회원가입 시도
3. 새로고침 (F5) 또는 자동 업데이트 확인
4. 새 사용자가 목록에 나타나는지 확인

==========================================
문제 해결
==========================================

Q: 사용자가 목록에 나타나지 않습니다
A:
   1. 페이지 새로고침 (F5)
   2. Vercel 함수 로그 확인:
      - Vercel → Deployments → 최신 배포 → View Function Logs
   3. 브라우저 콘솔(F12)에서 에러 확인
   4. Supabase 프로젝트가 활성화되어 있는지 확인

Q: 사용자는 있지만 "Confirmed"가 체크되지 않았습니다
A:
   1. Supabase "Authentication" → "Settings"
   2. "Enable email confirmations" 해제 확인
   3. 또는 사용자 상세에서 수동으로 확인 처리:
      - 사용자 클릭 → "Confirm email" 버튼

Q: 사용자가 여러 개 생성되었습니다
A:
   1. 중복 사용자 삭제:
      - 사용자 행 클릭
      - "Delete user" 버튼 클릭
   2. 또는 "..." 메뉴 → "Delete user"

==========================================
빠른 확인 체크리스트
==========================================

□ Supabase "Authentication" → "Users" 페이지 열기
□ 회원가입 시도
□ 페이지 새로고침 (F5)
□ 새 사용자가 목록에 나타나는지 확인
□ 사용자 이메일이 올바른지 확인
□ "Confirmed" 상태 확인
