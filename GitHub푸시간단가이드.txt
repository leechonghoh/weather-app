==========================================
GitHub 푸시 간단 가이드 (3단계)
==========================================

코드를 수정한 후 Vercel에 반영하려면 GitHub에 푸시해야 합니다!

==========================================
1단계: GitHub Desktop 열기
==========================================

- GitHub Desktop 프로그램 실행
- 좌측에서 weather-app 프로젝트 선택
- (없다면 "File" → "Add Local Repository"로 추가)

==========================================
2단계: 커밋 (Commit)
==========================================

- 하단 "Summary"에 설명 입력:
  예: "오류 수정" 또는 "회원가입 개선"
- "Commit to main" 버튼 클릭

==========================================
3단계: 푸시 (Push)
==========================================

- 상단 "Push origin" 버튼 클릭
- 완료!

==========================================
자동 배포 확인
==========================================

1. Vercel 대시보드 접속
2. 프로젝트 선택
3. "Deployments" 탭 확인
4. 새 배포가 자동으로 시작됨
5. "Ready" 상태가 되면 완료!

==========================================
끝!
==========================================

이제 배포된 사이트에서 변경사항을 확인할 수 있습니다!
