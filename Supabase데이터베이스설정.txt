==========================================
Supabase 데이터베이스 설정 가이드
==========================================

이 단계는 즐겨찾기 기능이 작동하기 위해 필수입니다!

==========================================
1단계: SQL Editor 열기
==========================================

1. Supabase 대시보드 접속
2. 프로젝트 선택
3. 좌측 메뉴에서 "SQL Editor" 클릭
4. "New query" 버튼 클릭

==========================================
2단계: 스키마 파일 찾기
==========================================

프로젝트 폴더에서 다음 파일을 찾으세요:
- c:\projects\weather-app\supabase\schema.sql

이 파일을 메모장이나 텍스트 에디터로 엽니다.

==========================================
3단계: SQL 실행
==========================================

1. schema.sql 파일의 전체 내용 선택 (Ctrl+A)
2. 복사 (Ctrl+C)
3. Supabase SQL Editor에 붙여넣기 (Ctrl+V)
4. 우측 하단 "Run" 버튼 클릭
   또는 키보드 단축키: Ctrl+Enter

==========================================
4단계: 성공 확인
==========================================

다음 메시지가 나타나면 성공:
- "Success. No rows returned"

에러가 나타나면:
- 에러 메시지를 확인
- SQL 문법 오류가 있는지 확인
- 다시 시도

==========================================
5단계: 테이블 확인 (선택사항)
==========================================

1. 좌측 메뉴 "Table Editor" 클릭
2. "favorites" 테이블이 보이는지 확인
3. 테이블 구조 확인:
   - id
   - user_id
   - city_name
   - country
   - query
   - created_at
   - updated_at

==========================================
완료!
==========================================

이제 데이터베이스가 준비되었습니다!
다음 단계로 진행하세요.
