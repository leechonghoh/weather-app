배포 체크리스트 (간단 버전)
==========================================

✅ 체크박스를 하나씩 확인하면서 진행하세요!

□ 1. GitHub 계정 만들기
   → https://github.com/

□ 2. GitHub에 코드 업로드
   → 새 저장소 만들기
   → 모든 파일 업로드

□ 3. Supabase 계정 만들기
   → https://supabase.com/
   → 새 프로젝트 생성
   → Project URL과 service_role key 복사

□ 4. OpenWeatherMap API 키 발급
   → https://openweathermap.org/api
   → 계정 만들기
   → API 키 복사

□ 5. Vercel에 배포
   → https://vercel.com/
   → GitHub로 로그인
   → 프로젝트 가져오기
   → 환경 변수 3개 설정:
     * SUPABASE_URL
     * SUPABASE_SERVICE_KEY
     * OPENWEATHER_API_KEY
   → Deploy 클릭

□ 6. 배포 완료 확인
   → 사이트 접속
   → 날씨 조회 테스트
   → 회원가입 테스트
   → 로그인 테스트
