# Weather Dashboard

전문적인 기업용 날씨 정보 대시보드 애플리케이션입니다.
도시 이름을 입력하면 해당 도시의 실시간 날씨 정보와 일주일 예보를 확인할 수 있습니다.

## 주요 기능

- 🌍 전 세계 도시의 날씨 정보 조회
- 🔍 실시간 자동완성 검색 기능
- ⭐ 즐겨찾기 기능 (localStorage 기반)
- 📊 일주일 날씨 예보 (카드형/그래프형 뷰)
- 💼 전문적인 기업용 UI 디자인
- 💻 데스크톱/노트북 최적화

## 기술 스택

- HTML5
- CSS3 (현대적인 디자인 시스템)
- Vanilla JavaScript (ES6+)
- OpenWeatherMap API
- SVG 아이콘

## 빠른 시작

### 1. OpenWeatherMap API 키 발급

1. [OpenWeatherMap](https://openweathermap.org/) 웹사이트에 접속
2. 회원가입 (무료 계정으로 충분)
3. 로그인 후 [API Keys](https://home.openweathermap.org/api_keys) 페이지에서 API 키 생성
4. 생성된 API 키 복사

### 2. API 키 설정

`script.js`와 `favorites.js` 파일의 상단에서 API 키를 설정하세요:

```javascript
const API_KEY = 'your_api_key_here';
```

### 3. 로컬 실행 (초보자를 위한 완전 상세 가이드)

> 💡 **코딩을 전혀 모르시는 분들을 위한 단계별 설명입니다.**
> 각 단계를 차근차근 따라하시면 됩니다.

⚠️ **중요**: 파일을 직접 더블클릭하는 대신 터미널에서 서버를 실행해야 합니다.
이유는 브라우저 보안 정책 때문입니다.

#### 🪟 Windows 사용자 (가장 쉬운 방법 - 추천!)

**📌 이 방법은 파일 탐색기에서 바로 실행할 수 있어 가장 쉽습니다.**

**1단계: weather-app 폴더 열기**
- 바탕 화면이나 다운로드 폴더에서 `weather-app` 폴더를 찾아서 더블클릭하세요
- 폴더가 열리면 안에 여러 파일들이 보일 겁니다

**2단계: start-server.bat 파일 찾기**
- 폴더 안에서 `start-server.bat` 파일을 찾으세요
- 파일 이름 옆에 타입이 "Windows 배치 파일"이라고 되어 있을 겁니다

**3단계: start-server.bat 파일 더블클릭**
- `start-server.bat` 파일을 더블클릭하세요
- 검은색 창(명령 프롬프트)이 뜨면서 여러 글자들이 나타날 겁니다
- 이것이 정상입니다! 서버가 시작되고 있는 것입니다

**4단계: 창 내용 확인**
- 검은색 창에 다음과 같은 글자들이 나타날 겁니다:
  ```
  ============================================================
  Weather Dashboard 서버가 시작되었습니다!
  ============================================================
  
  서버 주소: http://localhost:8000
  서버 주소: http://127.0.0.1:8000
  ```
- 이렇게 나오면 성공입니다!

**5단계: 웹 브라우저 열기**
- Chrome, Edge, Firefox 등 아무 웹 브라우저를 여세요
- 주소창(위쪽에 주소를 입력하는 곳)을 클릭하세요
- 다음을 정확히 입력하세요:
  ```
  http://localhost:8000
  ```
- 입력 후 Enter 키를 누르세요

**6단계: 날씨 앱 사용하기**
- 브라우저에 날씨 앱 화면이 나타나면 성공입니다!
- 이제 도시 이름을 입력해서 날씨를 검색할 수 있습니다

**7단계: 서버 종료하기 (사용을 마쳤을 때)**
- 검은색 명령 프롬프트 창을 클릭하세요
- 키보드에서 `Ctrl` 키와 `C` 키를 동시에 누르세요
- 창이 닫히거나 "서버가 종료되었습니다"라는 메시지가 나타나면 완료입니다
- **주의**: 서버를 종료하면 브라우저에서 날씨 앱이 작동하지 않습니다. 다시 사용하려면 3단계부터 다시 하세요.

#### 🪟 Windows 사용자 (Python이 설치된 경우)

먼저 Python이 설치되어 있는지 확인해야 합니다.

**1단계: PowerShell 열기**
- `Win` + `R` 키를 누르세요
- `powershell`을 입력하고 Enter

**2단계: Python 설치 확인**
- PowerShell에서 다음을 입력하고 Enter:
  ```
  python --version
  ```
- 버전 번호가 나오면 Python이 설치된 것입니다 (예: `Python 3.11.0`)
- "python을(를) 인식할 수 없습니다"라고 나오면 Python이 없는 것입니다
  → Python 설치: https://www.python.org/downloads/

**3단계: weather-app 폴더로 이동**
- PowerShell에서 다음을 입력 (여러분의 실제 경로로 바꿔주세요):
  ```
  cd "C:\Users\사용자이름\바탕 화면\weather-app"
  ```
- 또는 파일 탐색기에서 `weather-app` 폴더를 열고 주소창에 `powershell` 입력 후 Enter

**4단계: 서버 실행**
- 다음을 입력하고 Enter:
  ```
  python server.py
  ```

**5단계: 브라우저 열기**
- PowerShell에 "서버 주소: http://localhost:8000" 메시지가 나타나면
- 브라우저에서 `http://localhost:8000` 접속

**6단계: 서버 종료**
- `Ctrl` + `C` 키로 종료

#### 🍎 macOS 사용자

**1단계: 터미널 열기**
- `Cmd` + `Space` 키를 누르고 "터미널" 또는 "Terminal" 입력 후 Enter
- 또는 Finder > 유틸리티 > 터미널

**2단계: weather-app 폴더로 이동**
- 터미널에서 다음을 입력 (실제 경로로 바꿔주세요):
  ```
  cd ~/Desktop/weather-app
  ```
- 또는 Finder에서 `weather-app` 폴더를 터미널 창으로 드래그 앤 드롭 (경로가 자동으로 입력됨)

**3단계: 실행 권한 부여 (처음 한 번만)**
- 다음을 입력하고 Enter:
  ```
  chmod +x start-server.sh
  ```

**4단계: 서버 실행**
- 다음 중 하나를 입력:
  ```
  ./start-server.sh
  ```
- 또는 Python이 있다면:
  ```
  python3 server.py
  ```

**5단계: 브라우저 열기**
- 터미널에 "서버 주소: http://localhost:8000" 메시지가 나타나면
- 브라우저에서 `http://localhost:8000` 접속

**6단계: 서버 종료**
- `Ctrl` + `C` 키로 종료

#### ⚠️ 문제 해결

**"python을(를) 인식할 수 없습니다" 오류**
- Python이 설치되지 않았습니다
- https://www.python.org/downloads/ 에서 Python 설치
- 설치 시 "Add Python to PATH" 옵션을 반드시 체크하세요

**"포트가 이미 사용 중입니다" 오류**
- 다른 프로그램이 8000번 포트를 사용 중입니다
- `server.py` 파일을 열어서 `PORT = 8000`을 `PORT = 8080` 등으로 변경

**"파일을 찾을 수 없습니다" 오류**
- `cd` 명령어로 올바른 폴더에 있는지 확인하세요
- `pwd` (Linux/macOS) 또는 `cd` (Windows) 명령어로 현재 위치 확인

## 사용 방법

1. 도시 검색: 검색창에 도시 이름 입력 (2글자 이상 입력 시 자동완성)
2. 날씨 확인: 검색 결과에서 현재 날씨 정보 확인
3. 즐겨찾기: 별 아이콘 클릭하여 즐겨찾기 추가/제거
4. 상세 예보: "일주일 날씨 예보" 버튼 클릭하여 5일 예보 확인
5. 뷰 전환: 카드형/그래프형 뷰 전환 가능

## 주요 기능 설명

### 자동완성 검색
- 도시 이름을 입력하면 실시간으로 일치하는 도시 목록 표시
- 최대 10개 결과 표시
- 키보드 화살표로 선택 가능

### 즐겨찾기
- 브라우저 localStorage에 저장
- 즐겨찾기 페이지에서 모든 도시의 날씨를 한눈에 확인
- 개별 또는 전체 새로고침 가능

### 일주일 예보
- 5일치 날씨 예보 (3시간 간격)
- 카드형: 날짜별 상세 정보
- 그래프형: 온도 변화를 시각화한 연속 그래프

## 문제 해결

### "도시를 찾을 수 없습니다" 오류
- 도시 이름의 철자 확인
- 영어 도시는 영어로, 한국 도시는 한글로 입력

### "API 키가 유효하지 않습니다" 오류
- API 키가 올바르게 입력되었는지 확인
- OpenWeatherMap에서 API 키 활성화 확인 (발급 후 몇 분 소요될 수 있음)

### 날씨 정보가 표시되지 않음
- 인터넷 연결 확인
- 브라우저 콘솔(F12)에서 오류 확인
- API 호출 한도 확인

### CORS 오류
- HTTP 서버를 통해 실행해야 합니다 (file:// 프로토콜 사용 불가)
- `server.py` 또는 `server.js`를 사용하여 로컬 서버 실행
